# Face Recognition Attendance System

## Overview
This project is an AI-based attendance management system using face recognition technology.

## Features
- Real-time face detection
- Automated attendance marking
- Attendance stored in CSV files
- Student face registration

## Technologies Used
- Python
- OpenCV
- NumPy
- Pickle

## Project Workflow
1. Capture student face
2. Store face dataset
3. Match face in real-time
4. Automatically mark attendance

## Future Improvements
- GUI dashboard
- Database integration
- Web application support
